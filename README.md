# Guardiao 60+

Projeto inicial do produto Guardiao 60+ com foco em seguranca familiar para idosos.

## Estrutura

- `guardiao-60-plus-web`: app web publicado no GitHub Pages
- `docs`: PRD, arquitetura e roadmap do MVP
- `backend`: base para API de ingestao de dispositivos (fase de implementacao)

## Execucao local

```bash
cd guardiao-60-plus-web
npm install
npm run dev
```

## Publicacao

Deploy automatico no GitHub Pages pela pipeline em `.github/workflows/pages.yml`.
